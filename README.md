# 🚀 Prompt Engineering using LangChain

## 📌 Project Overview
This project is part of the **Gen-AI Internship (Feb 2026)** assignment.

The objective is to build a **Mini Prompt Engine** using LangChain by replacing hardcoded prompts with **dynamic, reusable PromptTemplates**.

---

## 🎯 Objectives
- Understand LangChain PromptTemplate
- Build dynamic prompt systems
- Replace hardcoded prompts
- Design reusable prompt pipelines
- Implement input validation

---

## 🛠️ Technologies Used
- Python 🐍
- LangChain 🔗
- Google Colab 📒

---

## Tasks Implemented

### Task 1: Basic Prompt Template
Converted hardcoded prompt into reusable template.

### Task 2: Multi-input Prompt System
Handled multiple inputs like topic, audience, and tone.

### Task 3: Prompt Variations Engine
Created different prompt styles:
- Teaching
- Interview
- Storytelling

### Task 4: ChatPromptTemplate System
Implemented role-based prompting using:
- System role
- User role

### Task 5: Input Validation Layer
Validated audience and tone inputs with default handling.

### Task 6: Prompt Generator App
Built a function to generate prompts dynamically based on style.

### Task 7: Template Reusability Test
Tested one template with multiple inputs to ensure reusability.

---

## Pipeline Flow
User Input → Validation → Prompt Template → Generated Output
